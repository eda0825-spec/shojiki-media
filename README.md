# shojiki-media

SHOJIKI ブランド (SH-J001 / SH-J002) の動画アセット CDN。

楽天 GOLD サーバーが HTTP Range Request 非対応で iOS Safari の動画 autoplay が失敗するため、ここに置いて GitHub Pages / jsDelivr 経由で配信する。

## CDN URL

- GitHub Pages: `https://eda0825-spec.github.io/shojiki-media/<path>`
- jsDelivr: `https://cdn.jsdelivr.net/gh/eda0825-spec/shojiki-media@main/<path>`

両方とも Range Request 完全対応。
